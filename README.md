## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Set up environment variables:
   ```bash
   export LANGCHAIN_API_KEY="your-api-key"
   export LANGCHAIN_TRACING_V2=true
   ```

3. Explore the notebooks in order
